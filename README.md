
# Active Directory Help Desk Home Lab

## Overview
Enterprise-style Active Directory lab demonstrating real Help Desk tasks.

## Lab Components
- Windows Server 2022 (Domain Controller)
- Windows 10/11 Client
- Active Directory Domain Services
- Group Policy
- PowerShell
